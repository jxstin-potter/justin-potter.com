# Justin Potter - Portfolio

A minimal portfolio website built with React and TypeScript.

## Tech Stack

- React 19.1.0
- TypeScript
- CSS3

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Scripts

- `npm start` - Development server
- `npm run build` - Production build
- `npm test` - Run tests
